Instructions
  1. Open the old inventory system's index.html in your browser.
  2. Press F12 to open Developer Tools, go to the Console tab.
  3. Copy and paste each command below, one at a time. Copy the output (the array string).
  4. Select the matching entity type below, paste the JSON, and click Import.
Step 1 — Copy from Old System

Run these in the old system's Console (press the Copy button):

Step 2 — Paste & Import Here
Order Matters!

Import in this order to avoid missing references:

  1. Raw Materials & Suppliers first (they have no dependencies)
  2. Products next (standalone)
  3. Purchases & Production & Sales last (they reference raw materials / products)

Duplicates (same name) are skipped automatically.
